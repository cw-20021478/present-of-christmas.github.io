# 크리스마스 선물 교환식 - 이름 뽑기 도구 🎄

크리스마스 선물 교환식을 위한 인터랙티브한 이름 뽑기 웹 애플리케이션입니다.

## ✨ 주요 기능

- **가챠 기계 스타일의 이름 뽑기**: 버튼을 누르면 가챠 기계에서 캡슐이 나와 이름을 공개합니다
- **중복 방지**: 한 번 뽑힌 이름은 다시 뽑히지 않습니다
- **참가자 관리**: 좌측 상단 버튼으로 참가자 목록을 쉽게 추가/삭제할 수 있습니다
- **자동 초기화**: 참가자 목록이 변경되면 뽑힌 목록이 자동으로 초기화됩니다
- **크리스마스 테마**: 크리스마스 분위기의 디자인과 애니메이션
- **반응형 디자인**: 모바일과 웹 환경 모두 지원
- **초기화 기능**: 실수로 뽑았을 경우 다시 시작할 수 있습니다

## 🎮 사용 방법

### 참가자 설정

1. 좌측 상단의 "👥 참가자 수정" 버튼을 클릭합니다
2. 팝업 창에서 참가자 이름을 입력하고 "추가" 버튼을 클릭합니다
3. 참가자를 삭제하려면 각 참가자 옆의 "삭제" 버튼을 클릭합니다
4. "저장" 버튼을 클릭하여 변경사항을 저장합니다

### 이름 뽑기

1. "🎁 이름 뽑기" 버튼을 클릭합니다
2. 추출구에서 빛이 나며 캡슐이 등장합니다
3. 캡슐이 커지고 위아래로 분리됩니다
4. 분리된 캡슐 사이에서 이름이 적힌 종이가 나타납니다
5. 모든 이름을 뽑을 때까지 반복합니다
6. 실수가 있다면 우측 상단의 "🔄 초기화" 버튼을 사용합니다

## 🛠️ 기술 스택

- Vanilla JavaScript

- 데스크톱 브라우저
- 모바일 브라우저 (iOS, Android)

- 크리스마스 테마의 아름다운 디자인
- 부드러운 애니메이션 효과
- 직관적인 사용자 인터페이스
- 눈 내리는 배경 효과
- 크리스마스 장식 요소 (리본, 별, 트리 등)

## 📝 참가자 설정

참가자 목록은 웹 인터페이스에서 직접 수정할 수 있습니다:

1. 좌측 상단의 "👥 참가자 수정" 버튼 클릭
2. 팝업 창에서 참가자 추가/삭제
3. "저장" 버튼으로 변경사항 저장

> **참고**: 참가자 목록을 변경하면 현재까지 뽑힌 이름들이 자동으로 초기화됩니다.

## 🚀 실행 방법

1. 저장소를 클론하거나 파일을 다운로드합니다
2. `index.html` 파일을 브라우저에서 엽니다
3. 바로 사용할 수 있습니다!
